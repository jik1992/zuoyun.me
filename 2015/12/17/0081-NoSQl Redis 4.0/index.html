NoSQl Redis 4.0

发表于 2015-12-17 | 分类于 tryghost |
字数统计: 926 | 阅读时长 ≈ 5

简介

kv 数据库， 单线程高并发，支持集群，可以实现队列，事务场景，有原子cas 类实现计数。
有几个概念，数据库，表

场景

  • 缓存-分布式缓存
  • 会话-分布式会话
  • 队列 -消息队列
  • 广播-订阅通知模型
  • 原子性操作-分布式锁、防超卖

优点

  • 支持5种数据类型string、hash、list、set、zset
  • 高性能 8万/s TPS
  • 集群 一致性hash的集群分布，支持从复制的高可用方案
  • 持久化机制 AOF、RDB
