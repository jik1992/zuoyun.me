安装

  • 前置安装 yum -y install unixODBC
  • 走 docker
    https://hub.docker.com/r/zabbix/zabbix-3.0/
  • 默认账号密码 Admin zabbix
  • 开启中文语言支持
  • 安装 agent => Configuration——Hosts——Create
