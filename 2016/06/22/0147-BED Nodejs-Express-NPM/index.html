强制包更新

方法一手动跟新：
修改package.json中依赖包版本，执行npm install –force
方法二使用第三方插件：
npm install -g npm-check-updates
ncu // 查看可更新包
ncu -u // 更新package.json
npm install // 升级到最新版本
方法三使用yarn代替npm,yarn upgrade
